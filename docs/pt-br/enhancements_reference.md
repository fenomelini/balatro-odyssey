# ✨ Lista de Aprimoramentos (Enhancements)

Os Aprimoramentos modificam cartas individuais do seu baralho, concedendo bônus permanentes. Diferente das Edições, os aprimoramentos são aplicados diretamente à "matéria" da carta.

| Nome | Efeito |
| :--- | :--- |
| **Carta de Madeira** | +10 Fichas. 1 em 8 chance de quebrar ao pontuar. |
| **Carta de Plástico** | +15 Fichas. |
| **Carta de Borracha** | Reativa esta carta 1 vez adicional. |
| **Carta de Cerâmica** | X1.5 Multi. 1 em 4 chance de quebrar ao pontuar. |
| **Carta de Tecido** | +5 Fichas e +2 Multi. |
| **Carta de Papel** | +30 Fichas. 1 em 6 chance de quebrar ao pontuar. |
| **Carta de Diamante** | Ganha $4 quando esta carta pontua. |
| **Carta de Rubi** | +10 Multi. |
| **Carta de Esmeralda** | +50 Fichas. |
| **Carta de Sombra** | X1.5 Multi. |
| **Carta de Luz** | 1 em 4 chance de ganhar $3 quando pontua. |
| **Carta de Planta** | Ganha permanentemente +5 Fichas quando pontua. |
| **Carta Sagrada** | +10 Multi e +30 Fichas. |
| **Carta Morta-Viva** | 1 em 3 chance de reativar esta carta. |
| **Carta Maldita** | X2 Multi e -20 Fichas. |
| **Carta Mágica** | 1 em 6 chance de criar uma carta de Tarô quando pontua. |
| **Carta de Platina** | +100 Fichas. |

---

### Notas Técnicas
*   **Quebra de Cartas**: As chances de quebra (Madeira, Cerâmica, Papel) são afetadas por probabilidades globais.
*   **Acúmulo**: Uma carta só pode ter um aprimoramento por vez.
