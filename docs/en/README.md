# Balatro Odyssey Documentation

Welcome to the official documentation for **Balatro Odyssey**, a massive expansion for Balatro adding over 1,000 unique Jokers, thematic decks, and cosmic-themed mechanics.

## Overview

Balatro Odyssey introduces complex cosmic-themed mechanics organized into 17 thematic groups:
- Singularity
- Quantum
- Temporal
- Dimensional
- And more...

## Content

This documentation covers:
- **Jokers**: Detailed reference for all 1,000 Jokers.
- **Decks**: Information on the 100 thematic decks.
- **Consumables**: Tarots, Planets, and Spectrals.
- **Mechanics**: Vouchers, Enhancements, Blinds, and Tags.
- **Guides**: Development and contribution guides.
