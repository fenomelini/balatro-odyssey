## ğŸš€ How to Play

1.  **Prerequisites**:
    - Ensure you have [Steamodded](https://github.com/Steamodded/smods) installed.
    - Ensure you have [Lovely](https://github.com/WilsontheWolf/lovely-injector) installed.
2.  **Download**:
    - Go to the [Releases](https://github.com/fenomelini/balatro-odyssey/releases) page and download the latest `BalatroOdyssey.zip`.
3.  **Installation**:
    - Locate your Balatro Mods folder (usually `%AppData%/Balatro/Mods` on Windows or `~/.local/share/Balatro/Mods` on Linux).
    - Extract the zip file so that you have a folder named `BalatroOdyssey` inside your `Mods` folder.
    - Inside `BalatroOdyssey`, you should see `BalatroOdyssey.lua`, `manifest.json`, and the `assets` folder.
4.  **Launch**:
    - Start Balatro and check the Mods menu to ensure Balatro Odyssey is active.
